Steps to configure on Redhat(rpm) based linux distros -
(Make sure python version =>3.8 and <3.11 is installed along with pip)



CentOS9 -
1. yum install mesa-libGL mesa-libEGL mesa-dri-drivers python3-qt5 gtk3-devel gnome-terminal dbus-x11 -y

2. git clone -b v1 https://github.com/ansys-internal/python-installer-qt-gui.git

3. cd python-installer-qt-gui

4. pip install pip -U

5. pip install -e .[freeze]

6. ansys_python_installer


RHEL8 - (must be registered to either redhat developer or premium subscription)

Make sure python3 is installed, if not then follow these steps -

1. yum update | dnf update

2. yum install python39 python39-setuptools -y

Steps to install python manager for DEV's - (Note: RHEL8 only has glibc2.28 as latest version, & this package depends on glibc2.29, need to install manually)

1. yum update | dnf update

2. yum install mesa-libGL mesa-libEGL fontconfig libxkbcommon libwayland-cursor libwayland-egl mesa-dri-drivers -y

3. yum install python3-qt5 gnome-terminal dbus-x11 -y
dnf copr enable mattrose/python3-terminator
dnf install terminator
yum install python3-configobj

4. git clone -b v1 https://github.com/ansys-internal/python-installer-qt-gui.git

5. cd python-installer-qt-gui

6. pip install pip -U

7. pip install -e .[freeze]

8. ansys_python_installer
